## 3. 首页视觉说明

- 仅 3 个入口按钮：
  1. 咪咪咪咪你一个小毛咪在这里
  2. 咪咪 姐姐这里有好吃的
  3. 咦 这个小毛咪看起来在生气
- 首页禁止直接出现：`哼！`、`哼。`、麻袋、火锅、奶茶、甜甜的。

## 4. 摸摸分支视觉说明

- `TOUCH_000` 展示三个触摸方向（脑壳/小毛手/毛肚肚）。
- `TOUCH_*_YES_*` 显示亲亲反馈与音频喇叭按钮。
- `TOUCH_*_NO_*` 走失败重选弹窗。

## 5. 投喂分支视觉说明

- `FOOD_000` 进入三个问题组：火锅、奶茶、甜甜的。
- 成功项走成功弹窗（带食物 icon 与“截图分享给姐姐”）。
- “盯（不理人）”走亲亲/抱抱/揉揉反馈弹窗。

## 6. 生气分支视觉说明

- `ANGRY_000` 只在该页展示：`哼~`、`哼！`、`哼。`。
- `哼~` 进入声音安抚组（嘿嘿/嘻嘻/桀桀）。

## 7. 哼哼分支视觉说明

- `HMPH_000` 进入三档需求识别：
  - `HMPH_100`（要摸摸）
  - `HMPH_200`（要抱抱）
  - `HMPH_300`（要亲亲）
- “不要”按钮走失败重选反馈。

## 8. 麻袋分支视觉说明

- `BAG_000` 仅从 `ANGRY_000` 进入。
- 三个麻袋（粉/蓝/紫）对应三个独立反馈结果弹窗。

## 9. 弹窗视觉说明

- 成功弹窗：白底圆角、可加柔粉点缀、按钮“截图分享给姐姐”。
- 失败弹窗：标题“可恶！”，文案“哎呀！被亲亲了！”，按钮“重选一次吧”。
- 反馈弹窗：亲亲 / 抱抱 / 揉揉三类。
- 音频弹窗：带喇叭按钮与播放占位反馈。
- 分享弹窗：保留“分享海报功能待接入”占位提示。
